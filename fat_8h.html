◆ file_access_t

enum file_access_t

File access mode flags.

Bitmask flags used when opening a file via fat_fopen(). Flags may be combined unless otherwise restricted.

Note
Mode validation is implementation-defined.
Enumerator
FILE_READ 

Open file with read permission.

FILE_WRITE 

Open file with write permission.

FILE_CREATE 

Create file; overwrite if exists.

FILE_CREATE_NEW 

Create file; fail if already exists.

FILE_APPEND 

Open file and set cursor to end.
